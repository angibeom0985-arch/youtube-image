캐릭터 생성

일관된 스타일의 캐릭터를 생성하여 전체 영상에서 동일한 인물을 유지할 수 있습니다.

스토리보드 생성

스크립트를 바탕으로 각 장면에 맞는 이미지를 자동 생성하여 영상 제작 시간을 단축합니다.

1

API 키 준비

Google Gemini API 키가 필요합니다. 아직 없으시다면:

API 키 입력

메인 페이지 상단의 "Google Gemini API 키" 입력란에 키를 붙여넣고 "설정 완료"를 클릭하세요.

3

기능 선택

캐릭터 생성 또는 스토리보드 생성 중 원하는 기능을 선택하여 사용하세요.

👤 캐릭터 생성 가이드

입력 방법

기본 정보

성별, 나이, 직업 등 캐릭터의 기본적인 특성을 입력하세요.

외모 특징

머리색, 헤어스타일, 키, 체형 등 구체적인 외모를 묘사하세요.

복장 스타일

옷차림, 악세서리 등 캐릭터가 착용할 의상을 설명하세요.

입력 예시

"20대 중반의 젊은 여성, 긴 흑발을 한쪽으로 넘긴 헤어스타일, 밝은 표정의 둥근 얼굴, 캐주얼한 흰색 셔츠와 청바지 착용, 자신감 있는 미소를 짓고 있음"

📋 스토리보드 생성 가이드

스크립트 작성 요령

장면 구분

각 장면을 명확히 구분하여 작성하세요. 장면 전환 지점을 표시하는 것이 좋습니다.

시각적 요소

등장인물의 행동, 배경, 소품 등 화면에 나타날 요소들을 구체적으로 기술하세요.

감정과 분위기

각 장면의 분위기나 등장인물의 감정 상태를 명시하여 더 생동감 있는 이미지를 생성하세요.

스크립트 예시

장면 1:

주인공이 아침에 침대에서 일어나며 창밖을 바라보고 있다. 따뜻한 햇살이 방 안으로 들어온다.

장면 2:

주방에서 커피를 내리며 생각에 잠긴 표정을 하고 있다. 스팀이 올라오는 커피잔이 클로즈업된다.

장면 3:

문을 나서며 결심한 듯한 표정으로 하늘을 올려다본다. 푸른 하늘과 구름이 보인다.

⚡ 고급 활용법

스타일 통일성

같은 프로젝트에서는 비슷한 스타일 키워드를 사용하여 일관된 느낌을 유지하세요.

배치 생성

여러 이미지를 한 번에 생성할 때는 각 설명을 명확히 구분하여 입력하세요.

품질 최적화

구체적이고 상세한 설명일수록 더 정확하고 고품질의 이미지가 생성됩니다.

🔧 자주 묻는 질문

Q: 이미지가 생성되지 않아요

A: API 키가 올바르게 입력되었는지 확인하고, 인터넷 연결 상태를 점검해주세요. 또한 Google AI Studio에서 API 사용량 한도를 확인해보세요.

Q: 생성된 이미지가 원하는 것과 다릅니다

A: 더 구체적이고 상세한 설명을 입력해보세요. 스타일, 색감, 구도 등을 명확히 지정하면 더 정확한 결과를 얻을 수 있습니다.

Q: 다운로드한 파일이 열리지 않아요

A: ZIP 파일을 완전히 다운로드한 후 압축을 해제해주세요. 브라우저의 다운로드 관리자에서 다운로드가 완료되었는지 확인하세요.

Q: 모바일에서도 사용할 수 있나요?

A: 네, 반응형 디자인으로 제작되어 모바일 기기에서도 정상적으로 사용할 수 있습니다. 다만 PC 환경에서 더 편리하게 이용하실 수 있습니다.
