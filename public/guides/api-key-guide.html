Google Gemini API 키 발급 가이드

유튜브 롱폼 이미지 생성기를 사용하기 위해 필요한 Google Gemini API 키를 발급받는 방법을 단계별로 안내드립니다.

중요사항

• API 키는 개인정보로 절대 타인과 공유하지 마세요.

• Google AI Studio는 무료로 제공되지만, 사용량 제한이 있습니다.

• API 키가 없으면 이미지 생성 기능을 사용할 수 없습니다.

1

Google AI Studio 접속

Google AI Studio 웹사이트에 접속하여 계정으로 로그인합니다.

접속 주소:

https://aistudio.google.com

💡 팁: Google 계정으로 로그인하면 됩니다. 별도 계정 생성이 필요하지 않습니다.

2

API 키 생성 페이지 이동

로그인 후 좌측 사이드바에서 "Get API key" 메뉴를 클릭합니다.

위 스크린샷처럼 좌측 사이드바에서 "Get API key" 버튼을 클릭하세요

찾는 방법:
좌측 메뉴 → "Get API key" 또는 "API 키 받기" → "Create API key" 버튼 클릭

💡

팁: 빨간색 화살표로 표시된 "Get API key" 버튼을 클릭하면 API 키 생성 페이지로 바로 이동할 수 있습니다.

3

새 API 키 생성

"Create API key in new project" 또는 기존 프로젝트를 선택하여 API 키를 생성합니다.

✅ 새 프로젝트 생성 (권장): "Create API key in new project" 선택

⚠️ 기존 프로젝트 사용: Google Cloud 프로젝트가 있는 경우에만 선택

4

API 키 복사 및 보관

생성된 API 키를 복사하여 안전한 곳에 보관합니다.

API 키 형태 예시:

AIzaSyB1234567890abcdefghijklmnopqrstuvwx

보안 주의사항

API 키를 절대 공개하지 마세요. 다른 사람이 악용할 수 있습니다.

5

웹사이트에서 API 키 사용

발급받은 API 키를 유튜브 이미지 생성기에 입력하여 사용합니다.

사용 방법:
1. 메인 페이지에서 "Google Gemini API 키" 입력란에 붙여넣기
2. "설정 완료" 버튼 클릭
3. 이제 모든 기능을 사용할 수 있습니다!

자주 묻는 질문 (FAQ)

Q: API 키가 작동하지 않아요

A: API 키를 정확히 복사했는지 확인하고, 앞뒤 공백이 없는지 체크해주세요. 또한 Google AI Studio에서 해당 프로젝트가 활성화되어 있는지 확인하세요.

Q: 무료로 얼마나 사용할 수 있나요?

A: Google Gemini API는 월 일정량까지 무료로 제공됩니다. 정확한 한도는 Google AI Studio에서 확인할 수 있습니다.

Q: API 키를 잃어버렸어요

A: Google AI Studio에서 새로운 API 키를 생성하거나, 기존 키를 다시 확인할 수 있습니다. 보안을 위해 이전 키는 비활성화하는 것이 좋습니다.
